# Компоненты

Компоненты — повторно используемые строительные блоки сценариев, хранящиеся в базе данных.

Используйте компоненты для фрагментов, которые нужно независимо редактировать,
версионировать и подключать повторно, вместо копирования YAML между сценариями.
Жизненный цикл и схема компонентов будут дополняться здесь по мере стабилизации модели.

# Диагностика неполадок

## Логи приложения

```bash
make logs
```

## Состояние контейнеров

```bash
make ps
docker compose config
```

Добавляйте сюда повторяющиеся инциденты после того, как установлена первопричина
и проверены шаги восстановления. Не превращайте страницу в неотфильтрованный дамп логов.

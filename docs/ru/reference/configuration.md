# Конфигурация

Скопируйте `.env.example` в `.env` и замените значения-заглушки на параметры целевого
окружения. Не храните учётные данные и адреса конкретных окружений в опубликованных
описаниях сценариев.

Стандартное развёртывание использует PostgreSQL и шину в памяти процесса. Шину на базе
NATS можно включить через предусмотренный профиль Compose и переменную `BUS_BACKEND`.

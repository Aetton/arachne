# Архитектурные решения

В этом разделе фиксируются значимые архитектурные решения Arachne: контекст,
рассмотренные варианты, принятое решение и его последствия.

Для каждого решения создавайте отдельную Markdown-страницу. Не переписывайте старые
решения задним числом: если архитектура изменилась, добавьте новое решение со ссылкой
на заменённое.

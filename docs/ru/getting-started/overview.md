# Обзор

Arachne — лёгкий портал оркестрации CI/CD-сценариев и автоматизации инфраструктуры.
Оператор выбирает сценарий, заполняет параметры и запускает его. Arachne проверяет
права доступа, передаёт настроенные шаги плагинам-драйверам, транслирует логи и
собирает ссылки на артефакты.

## Основные понятия

- **Сценарий (scenario)** — версионируемое описание рабочего процесса.
- **Компонент (component)** — повторно используемый строительный блок сценария.
- **Драйвер (driver)** — адаптер для системы выполнения.
- **Триггер (trigger)** — условие ручного, запланированного или цепочечного запуска.
- **Шина (bus)** — передаёт запросы и ответы между порталом и воркерами.
